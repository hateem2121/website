The paper & ink design system
The Extra Mile  |  For a Better Tomorrow  |  Never Look Back
One identity in two lighting conditions — light = "The Workshop" (warm paper, ink type, volt stamps) and dark = "The Night Shift" (deep olive, off-white type, volt glowing) — with a technical-drawing feel running through both. This is a portable copy for use anywhere. The authority is docs/design/DESIGN.md; if they differ, that file wins.

Colours

N°001

Copy the hex codes exactly — never re-pick by eye. All text/background pairs are contrast-verified (readable for low-vision users).

The signal colour — same in both modes

Volt #CDF345
THE accent. Buttons, pills, highlights, stamps. Never as small text on paper — unreadable there.
Volt Deep #5F7414
Use whenever volt-coloured TEXT or thin lines sit on a light background. Readable where plain volt is not.

Shape, grid & motion

N°003

Corners

  • Cards / panels ≈ 18px
  • Buttons / pills ≈ 8px
  • Nav pills — fully round

Signature texture

  • Faint blueprint grid (like the background of this page)
  • Garment wireframe drawings for empty / loading / placeholder states
  • One dramatic full-width band per page, max

Movement

  • Precise, unhurried, alive
  • Quick reactions + soft settles
  • Always honour a device's "reduce motion" setting

The "don't break these" list

N°006
  • 01Use the exact hex codes above — don't re-pick by eye.
  • 02Volt is a signal, never the whole design. It accents a paper-and-ink system.
  • 03Never volt text on paper (unreadable) — use Volt Deep for volt-coloured text on light.
  • 04Two font families max; Archivo everywhere until the Neue Stance licence is confirmed.
  • 05Light and dark are both real designs — never invert one to make the other.
  • 06B2B voice only: "Request a Quote," never "buy / order."
  • 07Never say RUN itself is certified.
  • 08Respect "reduce motion."
  • 09In doubt? docs/design/DESIGN.md is the authority; locked values change only on Hateem's call.
