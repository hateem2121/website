06 Motion Character Kinetic type · hard cuts over eases

Motion behaves like a printing press: type slides into place in hard 100–200 ms cuts, stepped rather than eased — a mechanical snap, never a float.

Headlines reveal line by line as if the press is laying them down. Stat strips run as tickers (1889 → 2026, 100,000 units a month). Image blocks swap like turning pages — an instant wipe, no crossfade. Buttons snap up-and-left with a hard offset shadow, like a stamp lifting off paper.

  • BoldHome hero, heritage story, category landing pages — full kinetic type and tickers.
  • SubtleRFQ forms and buyer portal — 100 ms opacity and border shifts only. Nothing moves that the buyer is trying to read or fill.
  • MinimalCatalog grids and spec tables — no entrance animation; hover states snap instantly.
  • AlwaysEvery animation sits inside prefers-reduced-motion: no-preference. Reduced motion gets the finished layout, static.
